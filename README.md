# Advanced Math Learning Platform

A comprehensive web application for learning mathematics from beginner to advanced levels.

## Features

- Interactive lessons for various math topics
- Calculators and equation solvers
- Visualizations and graphs
- Progressive learning path
- Quizzes and exercises

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Topics Covered

- Arithmetic
- Algebra
- Geometry
- Calculus
- Statistics (coming soon)

## Technologies Used

- Next.js
- React
- TypeScript
